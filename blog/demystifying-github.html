So, you're new to programming and maybe heard from a friend or a youtube video about something called Github. Well, let me tell ya, it's pretty neat (& neat definitely an understatement!)

A quick overview: Github is a web-based version control system for tracking changes in files and coordinating work on projects. It's a great way to share code and collaborate on projects.

GitHub has become one of the few necessary platforms to use in today's web development world.
